Nasdaq Stock web-scraper


Scrapes and cleans data from of NASDAQ stocks and related data from Yahoo Finance; data is exported to CSV(file is used as a database). In particular, when it cleans data, it only accesses yahoo URLs that exist. This is achieved through dynamic link creation; it’s a highly scalable project, which allows for easy search and analysis of stock data via MS Excel. This data can also be exported to XML formats to be used in SQL.
